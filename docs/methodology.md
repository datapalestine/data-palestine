# Methodology

## Our Approach

Data Palestine collects, processes, and publishes Palestinian statistical data following FAIR principles: Findable, Accessible, Interoperable, and Reusable.

## Data Sources

All data is sourced from official and authoritative institutions. See the full list of sources on the platform.

## Data Processing Pipeline

1. **Collect** — Download raw files from source websites
2. **Extract** — Parse tables from HTML, PDF, and Excel files
3. **Transform** — Normalize schemas, clean values, validate
4. **Load** — Insert into database with full provenance tracking

## Quality Assurance

Every data point is validated against its source and linked to the original document.

## Limitations

Data availability varies by region, time period, and indicator. Some historical data may be incomplete.

## Transparency Commitment

All processing code is open source. Every observation traces back to its original source document.
