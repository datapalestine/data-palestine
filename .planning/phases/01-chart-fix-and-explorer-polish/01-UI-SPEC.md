# Phase 1 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none — hand-rolled components with Tailwind CSS v4 |
| Preset | not applicable |
| Component library | none (custom primitives in `apps/web/components/`) |
| Icon library | none currently (use text labels, not icon-only buttons) |
| Font | IBM Plex Sans (English) + IBM Plex Arabic (Arabic) via `--font-sans` CSS variable |

**No shadcn:** This is a brownfield Next.js 15 project with an existing bespoke component system using Tailwind CSS v4 `@theme` tokens. Introducing shadcn is out of scope for this phase. All new components use existing `globals.css` tokens directly.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Legend swatch gap, inline icon padding |
| sm | 8px | Legend item internal padding, chart margin top/right |
| md | 16px | Chart container horizontal padding, banner content padding |
| lg | 24px | Chart block vertical gap (legend → chart → attribution) |
| xl | 32px | Banner vertical margin from page top |
| 2xl | 48px | Not used in this phase |
| 3xl | 64px | Not used in this phase |

Exceptions: `ResponsiveContainer` height is fixed at 320px (not a spacing scale value — it is a chart dimension). Touch targets for banner dismiss button: 44px minimum.

## Typography

All type uses `font-family: var(--font-sans)` (IBM Plex Sans / IBM Plex Arabic). Western Arabic numerals (0–9) in both locales.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body (table cells, filter labels) | 14px | 400 | 1.5 |
| Label (axis ticks, legend names, attribution) | 11px | 400 | 1.4 |
| Heading (chart section headings, banner title) | 16px | 600 | 1.3 |
| Display | Not used in this phase | — | — |

Specific overrides:
- Axis tick labels (`XAxis`, `YAxis`): 11px, fill `#757575`, `fontFamily: "inherit"`
- Custom tooltip header (period label): 11px, `#757575`, weight 400
- Custom tooltip value: 13px, `#212121`, weight 600
- Legend item names: 11px, `#4B5563` (Tailwind `text-neutral-600` equivalent), truncated at 180px with `title` attribute for full name
- Disclaimer banner body text: 14px, weight 400
- Overflow badge text ("Showing 5 of N"): 12px, `#757575`, weight 400

## Component Contracts

### ChartLegend (new component)

**Location:** `apps/web/components/charts/ChartLegend.tsx`

**Purpose:** Replaces the Recharts built-in `<Legend />` which overflows and collapses the chart area. Rendered as a sibling div *above* `<ResponsiveContainer>`, never inside the Recharts tree.

**Visual contract:**
- Container: `flex flex-wrap gap-x-4 gap-y-1 px-4 pb-2` with no max-height constraint
- Each item: `flex items-center gap-1.5 min-w-0`
- Color swatch: `inline-block h-2.5 w-2.5 shrink-0 rounded-sm`, `backgroundColor: {color}`
- Name: 11px, `#4B5563`, truncated at `maxWidth: 180px` with CSS `truncate`, full name in `title` attribute
- No hover states, no click interactions (Phase 2)

### Overflow badge

**Location:** Inline below `<ChartLegend />`, above `<ResponsiveContainer>`

**Trigger:** Renders only when `selectedIndicators.length > MAX_CHART_SERIES` (5)

**Copy:** `Showing 5 of {N} indicators — select fewer to compare all`

**Visual contract:** 12px, `#757575`, no background, no border, `mt-1 px-4`

### Chart container layout

```
[ChartLegend]
[Overflow badge — conditional]
[ResponsiveContainer height=320]
[Source attribution]
```

**ResponsiveContainer:** `width="100%"` `height={320}`. Maintain existing `dynamic()` import pattern — do not use the Recharts 3.8 `responsive` prop until SSR behavior is confirmed.

### LineChartView and BarChartView (refactored)

**Location:** `apps/web/components/charts/` (split from `DataExplorer.tsx`)

**Series input:** Receives `series: string[]` pre-capped at 5. Never caps internally — cap happens in `DataExplorer` before passing.

**Recharts configuration (applies to both):**
```tsx
// CartesianGrid
strokeDasharray="3 3"  stroke="#E0E0E0"  vertical={false}

// XAxis
tick={{ fontSize: 11, fill: "#757575", fontFamily: "inherit" }}
tickLine={false}
axisLine={{ stroke: "#E0E0E0" }}
tickMargin={8}

// YAxis
tick={{ fontSize: 11, fill: "#757575", fontFamily: "inherit" }}
tickLine={false}
axisLine={false}
width={56}
niceTicks="nice"
tickFormatter={formatAxisValue}  // K/M/B abbreviation

// Chart margin
margin={{ top: 8, right: 16, bottom: 4, left: 0 }}

// Animations — ALL chart elements
isAnimationActive={false}  // on every <Line> and <Bar>

// No <Legend /> in Recharts tree — removed entirely
```

**formatAxisValue contract:**
```
>= 1B  → "{n}B" (1 decimal)
>= 1M  → "{n}M" (1 decimal)
>= 1K  → "{n}K" (0 decimal)
< 1 and > 0  → "{n}" (2 decimal)
else  → locale string, 0 decimal
```

**Custom tooltip:**
- Background: `#FFFFFF`, border: `1px solid #E0E0E0`, border-radius: 4px, no shadow
- Period label: 11px, `#757575`, weight 400, displayed at top
- Each series row: colored dot (8px circle, series color) + indicator name (11px, truncated 20 chars) + value (13px, `#212121`, weight 600)
- Cursor: `{ stroke: "#E0E0E0", strokeWidth: 1 }`

**Bar chart specific:** `barSize={24}` for single series; `barSize={16}` for 2–5 series (prevents bars touching). Bar `radius={[2, 2, 0, 0]}` (2px top corner rounding).

### Source attribution

**Location:** Static `<p>` rendered below `<ResponsiveContainer>`, inside the chart card

**Copy:** `Source: {datasetDetail.source_name}` (English) / `المصدر: {datasetDetail.source_name}` (Arabic)

**Visual contract:** 11px, `#757575`, weight 400, `mt-2 px-4 pb-3`

### Development disclaimer banner (new component)

**Location:** `apps/web/components/data/ExplorerDisclaimer.tsx`

**Placement:** Top of Data Explorer page, above the three-column layout, below the page heading. Full width.

**Persistence:** Dismissed state stored in `sessionStorage` (key: `explorer_disclaimer_dismissed`). Dismissed for the rest of the browser session. Reappears on new sessions.

**Visual contract:**
- Container: full-width, `background: #FFFDE7`, `border: 1px solid #F9A825`, `border-radius: 6px`, `padding: 12px 16px`
- Layout: `flex items-start justify-between gap-4`
- Icon: None. Text only.
- Title: "Data Explorer — Early Access" — 14px, weight 600, `#212121`
- Body: 14px, weight 400, `#212121`, `mt-1`
- Dismiss button: "×" character (not an icon), 20px, `#757575`, `aria-label="Dismiss notice"`, touch target 44×44px via padding, positioned top-right of banner
- No animation on dismiss (removes from DOM immediately via React state)

**Copy contract (English):**

## Interaction Contracts

### Chart cap behavior

- `MAX_CHART_SERIES = 5` constant defined in `DataExplorer.tsx` (or extracted to `lib/constants.ts`)
- Cap applied at `visibleSeries = seriesNames.slice(0, MAX_CHART_SERIES)` — not at fetch time
- Table view always shows all observations regardless of cap
- When user deselects indicators below 5, all remaining are shown (no "hidden" state below cap)

### Disclaimer dismissal

- Banner visible on initial load if `sessionStorage.getItem("explorer_disclaimer_dismissed")` is falsy
- On dismiss click: `sessionStorage.setItem("explorer_disclaimer_dismissed", "1")` + React state removes banner from DOM
- No fade animation. Instant removal.
- Keyboard: dismiss button is focusable, triggered by Enter/Space

### Auto-select behavior change

- Current behavior (remove): on dataset load, auto-selects ALL indicators and triggers fetch
- New behavior: on dataset load, auto-selects the first 3 indicators by `sort_order` (not all)
- Rationale: prevents the chart cap from immediately showing "Showing 5 of N" on every load
- This change is in `DataExplorer.tsx` lines 190–199 where `allInds` is computed — change `allInds` to `detail.data.indicators.slice(0, 3).map(...)`

### Recharts Tooltip visibility

- Tooltip shows on hover (existing behavior preserved)
- No click-to-lock tooltip behavior (Phase 2)
- Custom content replaces default Recharts tooltip styling

## Accessibility Constraints

- Disclaimer dismiss button: `aria-label="Dismiss notice"` (both locales), minimum 44px touch target
- Chart color palette: no adjacent series are red+green (positions 1 and 5 are never adjacent in legend order)
- Overflow badge uses plain text (not color-only) to communicate truncation
- All chart text (axis labels, legend, attribution) must meet 4.5:1 contrast on white: `#757575` on `#FFFFFF` = 4.48:1 (passes AA for 11px bold — enforce weight 500+ at 11px if AA strict compliance is required; otherwise acceptable for UI text)
- Source attribution is always visible below chart (not inside SVG) so screen readers can reach it
- Banner title is `<p>` with weight 600, not `<h>` tag (does not disrupt heading hierarchy)

---

## RTL Contract

- Disclaimer banner layout: `flex-direction` is `row` in LTR, `row-reverse` in RTL (or use `dir="auto"` on container)
- Dismiss button: in RTL, dismiss button sits on the left side (start of reading direction)
- Chart axis direction: XAxis `reversed={locale === "ar"}` is NOT applied in this phase (deferred — needs testing). Charts render LTR in both locales for Phase 1.
- Legend: `flex-wrap` handles RTL naturally — no explicit override needed
- All i18n strings go through `next-intl` `useTranslations("explore")` hook
